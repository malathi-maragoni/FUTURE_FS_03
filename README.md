# ☕ The Mala Blend Cafe Website

A modern cafe website built as part of Future Interns Task 3.

## 🚀 Features
- Premium UI design
- Menu with cart system
- WhatsApp ordering
- Customer reviews
- Google Maps integration
- Open/Closed status

## 💡 Purpose
This website helps local cafés:
- Improve online presence
- Attract customers
- Increase orders via WhatsApp
